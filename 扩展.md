# webrc 部署
基于 WebRTC 协议的实时通信系统，核心包括：信令服务器（Signaling）、媒体流采集 / 处理、P2P 连接管理（含 NAT 穿透）等

```bash
# webrtc 部署
sudo apt install -y libwebrtc-audio-processing-dev
```

1. 基础依赖：
``` bash
运行
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装媒体处理工具（采集/编码）
sudo apt install -y gstreamer1.0-tools gstreamer1.0-plugins-base gstreamer1.0-plugins-good \
                     gstreamer1.0-plugins-bad gstreamer1.0-libav ffmpeg

# 安装信令服务器依赖（以 Node.js 为例）
sudo apt install -y nodejs npm  # Ubuntu 24.04 源中 Node.js 版本足够新

# 安装 TURN 服务器（用于 NAT 穿透，可选）
sudo apt install -y coturn
```

# 搭建信令服务器（Signaling Server）

```bash
# 创建项目目录
mkdir webrtc-signaling && cd webrtc-signaling

# 初始化 npm 项目
npm init -y

# 安装 Socket.io（WebSocket 库，用于实时通信）
npm install socket.io
```

## 编写信令服务器代码(Cpp)

首先安装 Boost 库(含 Boost.Beast)
```bash
sudo apt update
sudo apt install -y libboost-all-dev  # 包含 Boost.Asio、Boost.Beast 等
sudo apt install -y g++ cmake  # 编译工具
```
